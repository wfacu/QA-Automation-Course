## Calculadora Simple

### Consigna

1. Crear una calculadora por consola con opciones: sumar, restar, multiplicar y dividir.  
2. Validar que la opción ingresada sea correcta antes de continuar.  
3. Pedir dos números, realizar la operación y mostrar el resultado.  
4. Evitar división por cero mostrando un mensaje adecuado.  
5. Permitir repetir operaciones.  
6. Trabajar con una lista de alumnos que incluya nombre, edad, sexo y nota.  
7. Mostrar los datos de cada alumno en una sola línea.  
8. Calcular el promedio de las notas del grupo.  
9. Indicar cuántos alumnos tienen 18 años o más.  
10. Mostrar cuál es el alumno con la nota más alta.

## Sistema de Alumnos

### Consigna

1. Crear una lista llamada `alumnos`.  
2. Cada alumno debe ser un diccionario con las claves:  
   - `nombre`  
   - `edad`  
   - `notas` (lista)  
   - `datos_fijos` (tupla con DNI y año de ingreso)  
3. Cargar al menos 3 alumnos con datos de ejemplo.  
4. Mostrar todos los alumnos recorriendo la lista.  
5. Calcular y mostrar el promedio de notas de cada alumno.  
6. Determinar y mostrar el alumno con mayor promedio.  
7. Permitir agregar una nueva nota a un alumno.  
8. Ordenar los alumnos por promedio de mayor a menor.  
9. Mostrar alumnos con promedio mayor o igual a 7.
